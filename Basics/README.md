# Basics

This folder contains beginner-level Java programs used to build a strong foundation for Data Structures and Algorithms.

## Programs

* Largest of 3 Numbers
* Odd/Even Number
* Prime Number Check
* Reverse a Number
* Count Digits
* Sum of Digits
* Palindrome Number

These programs help in understanding:

* Conditional Statements
* Loops
* User Input
* Basic Number Operations
* Problem-Solving Logic
